## PRD (Product Requirements Document)

### Core Features

1. **Directory Tree Walk**
   - Accepts optional path (defaults to current directory)
   - Accepts `--depth N` flag (default: 2, max: 10)
   - Skips: `.git`, `node_modules`, `vendor`, `dist`, `build`, `__pycache__`, `.next`, `.kilo`
   - Returns file count, directory count, total size

2. **Expandable Tree Navigation**
   - Initial view: collapsed to root level at specified depth
   - Enter expands/collapses directory
   - Show chevron indicator for directories
   - Indentation: 2 spaces per level

3. **File Preview Panel**
   - Shows on file selection (not directories)
   - Displays first 50 lines
   - Syntax highlighting for known languages
   - Shows file size and last modified date

4. **Keyboard Navigation**
   - `j` / `↓`: Move down
   - `k` / `↑`: Move up
   - `l` / `→`: Expand directory
   - `h` / `←`: Collapse directory
   - `Enter`: Toggle expand/collapse
   - `q` / `Esc`: Quit
   - `g` / `G`: Go to start/end
   - `/`: Search/filter files

5. **Mouse Support**
   - Single click: Select item
   - Double click: Expand/collapse directory
   - Scroll: Navigate list

6. **Chat Integration**
   - `/explore [path]` - Show tree in chat
   - `/explore --interactive` - Launch interactive TUI
   - Auto-detect language and show emoji

### Edge Cases
- Empty directories: Show "(empty)" placeholder
- Permission denied: Skip with warning
- Symlinks: Show but don't recurse
- Very deep paths (>1000 files): Paginate, lazy-load children
- Binary files: Show file info without content preview
